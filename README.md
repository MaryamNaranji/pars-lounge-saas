# Pars Lounge Café – SaaS Starter

Next.js 14 + TypeScript + Tailwind + Stripe + Firebase + NextAuth.

## One-click deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/YOUR_GITHUB/pars-lounge-saas)

## Local dev

1. Clone repo  
2. `npm install`  
3. Copy `.env.local.example → .env.local` and fill keys  
4. `npm run dev`
